# Start a focus session

Pick what you want to work on. FocusFlow will break it into small steps so it’s easier to start.

## Choose a task (example)

**Selected task:** Finish lab report draft

**AI breakdown (editable):**
1. Open the report document
2. Write a 5-bullet Methods outline
3. Fill in 2–3 key details per bullet
4. Drop in 1 figure/table placeholder
5. Write a rough Results paragraph (no polishing)

**Starting step (tiny):** Open the report document and write the Methods section title.

## If I'm already behind today (example)
If you didn’t follow the original plan, the app can “shrink and reshuffle” instead of piling up guilt.

**AI suggestion:** Keep only 1 session goal for today, move the rest to tomorrow, and start with the smallest step now.
